Ikea Mood Forecast

Stack: jQuery, Web Audio API, CSS

Unfortunately, it was the only project without NDA when I worked in Smarthead. We created a lot of complex and beautiful solutions for famous brands, but I can't mention them.

Ikea Mood Forecast was a promo web application where room interior changes depending on the weather outside. By default it shows your city, but you can select any city in the world. And you can but everything in the photo on IKEA website.

Depending on the forecast window view also changes. On cloudy days you can see some clouds, on wet weather snow or rain animation. (In some interiors there was also a mirror with window reflection).

Also we had special music for this. It consisted of 5 streams, where every one of them showed state of temperature, humidity, wind speed, precipitations and atmosphere pressure. Default HTML5 audio couldn't keep the rhythm and sync 5 streams, so we used Web Audio API (but for old browsers like IE9 we still added HTML5 audio for backward compatibility).
